Normal CSS:

When using normal CSS, you have complete control over the design and layout of your website. You write all the CSS code yourself, which allows for maximum flexibility and customization.

Bootstrap:

Bootstrap is a popular front-end framework that provides pre-written, pre-styled CSS and JavaScript components. It's essentially a collection of CSS and JavaScript files that you can use as a starting point for your website.
